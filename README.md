### algorithm-group-study

#### 작업방법
1. fork
2. 자신의 레포지토리에서 clone 받아서 작업
3. 자신의 레포지토리에 작업내용 push (최초에는 자신의 깃헙 이름(BeomsooKim96)의 폴더 하나 생성해주세요 )
4. pull-request 생성
5. 원본에 merge
